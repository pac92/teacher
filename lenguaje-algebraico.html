¿Qué es el lenguaje algebraico?

Aprendemos a traducir palabras en expresiones matemáticas usando letras y símbolos.

Variables y constantes

En álgebra usamos letras para representar números desconocidos o que pueden cambiar.

x, y, a, n, k, …
Ejemplo: "Un número cualquiera" → n  |  "El doble de un número" → 2n

Una constante es un número fijo (como 3, 7, ½). Una variable es la letra que puede tomar distintos valores.

Expresión algebraica vs. Ecuación

Expresión: no tiene signo = →  3x + 5
Ecuación: sí tiene signo = →  3x + 5 = 20

Propiedades útiles

Conmutativa:  a + b = b + a  — el orden no cambia el resultado.
Asociativa:  (a + b) + c = a + (b + c)
Elemento neutro:  a + 0 = a
✏️
Practica: "María tiene n caramelos y le regalan 15 más" → n + 15

Fracciones algebraicas

División expresada como fracción: numerador entre denominador.

Estructura de una fracción algebraica

Numerador / Denominador  →  p / q  (q ≠ 0)
Frase verbalExpresión
La mitad de nn / 2

Operaciones con fracciones algebraicas

Suma / Resta → mismo denominador
a/n + b/n = (a + b) / n
Ejemplo:  x/4 + 3/4 = (x + 3)/4
Suma / Resta → distinto denominador
a/m + b/n = (an + bm) / mn
Ejemplo:  x/2 + x/3 = 3x/6 + 2x/6 = 5x/6
%

Porcentajes algebraicos

"Por ciento" significa "de cada 100". Convertimos el % en fracción o decimal.

Conversión fundamental

p% de x  →  (p/100) · x
Frase verbalExpresión algebraica
El 10% de x0.1x  =  x/10
El 25% de n0.25n  =  n/4
El 50% de m0.5m  =  m/2
El p% de una cantidad x(p/100)·x
Aumentar x en un r%x + (r/100)·x = x(1 + r/100)
Disminuir x en un r%x − (r/100)·x = x(1 − r/100)

Ejemplo contextualizado

Situación: Un artículo cuesta p euros y tiene un descuento del 15%.
Precio final: p − 0.15p = 0.85p
Situación: Una población de n personas crece un 3% cada año.
Después de 1 año: n · 1.03
