¿Qué es una ecuación
de primer grado?

Una igualdad con una incógnita elevada a la primera potencia.

Definición

Una ecuación de primer grado (o ecuación lineal) es una igualdad matemática en la que aparece una incógnita —habitualmente x— con exponente 1.

ax + b = c    (a ≠ 0)

Resolver la ecuación significa encontrar el valor de x que hace verdadera la igualdad. A ese valor se le llama solución o raíz.

Partes de una ecuación

TérminoQué es
Miembro izquierdoTodo lo que está a la izquierda del =
Miembro derechoTodo lo que está a la derecha del =
IncógnitaLa letra cuyo valor buscamos (p.ej. x)
CoeficienteEl número que multiplica la incógnita
Término independienteEl número sin letra

¿Es ecuación o expresión?

EjemploTipo
3x + 5Expresión ✗
3x + 5 = 20Ecuación ✓
x² + 1 = 02º grado ✗
2x − 1 = 7Ecuación ✓
⚠️ Si la incógnita aparece al cuadrado (x²) ya no es de primer grado.

La metáfora de la balanza

Una ecuación es como una balanza en equilibrio. Lo que hagas en un lado, debes hacerlo en el otro para mantener la igualdad.

Propiedades de las ecuaciones

Las herramientas que usamos para despejar la incógnita.

Principio de igualdad

Podemos realizar cualquier operación en ambos miembros sin romper la igualdad:

OperaciónReglaEjemplo
SumaSi a = b → a + c = b + cx − 3 = 7 → x = 10
RestaSi a = b → a − c = b − cx + 5 = 9 → x = 4
MultiplicaciónSi a = b → a·c = b·cx/3 = 4 → x = 12
DivisiónSi a = b → a/c = b/c2x = 8 → x = 4

Transposición de términos

Cuando pasamos un término al otro lado del signo =, cambia de signo. Es una consecuencia directa del principio de igualdad.

1
Ecuación original: x + 4 = 10
2
Pasamos el +4 al otro lado: x = 10 − 4
3
Resultado: x = 6
💡 Lo que suma pasa restando, y lo que multiplica pasa dividiendo.

Verificación de la solución

Siempre podemos comprobar si nuestra solución es correcta sustituyendo el valor en la ecuación original.

1
Ecuación: 2x − 3 = 7  →  solución obtenida: x = 5
2
Sustituimos: 2·(5) − 3 = 10 − 3 = 7 ✓
3
La igualdad se cumple, x = 5 es correcto.

Ecuaciones con suma y resta

El tipo más sencillo: solo hay que trasladar términos.

Tipo: x + b = c

x + b = c  →  x = c − b
1
Ecuación: x + 7 = 15
2
Pasamos el +7: x = 15 − 7
3
Solución: x = 8

Tipo: x − b = c

x − b = c  →  x = c + b
1
Ecuación: x − 4 = 9
2
Pasamos el −4: x = 9 + 4
3
Solución: x = 13

Incógnita en el miembro derecho

b = x + c  →  x = b − c
1
Ecuación: 12 = x + 5
2
Pasamos el +5: 12 − 5 = x
3
Solución: x = 7

Con términos en ambos miembros

x + 3 = 2x − 5  →  agrupa las x a un lado
1
Ecuación: x + 3 = 2x − 5
2
Pasamos la 2x: x − 2x = −5 − 3
3
Simplificamos: −x = −8
4
Multiplicamos por −1: x = 8
// sección 04

Ecuaciones con multiplicación y división

Cuando la incógnita lleva un coeficiente o está en el denominador.

Tipo: ax = c

ax = c  →  x = c / a
1
Ecuación: 3x = 21
2
Dividimos por 3: x = 21 / 3
3
Solución: x = 7

Tipo: x / a = c

x / a = c  →  x = c · a
1
Ecuación: x / 4 = 6
2
Multiplicamos por 4: x = 6 · 4
3
Solución: x = 24

Coeficiente fraccionario: (a/b)x = c

(a/b)x = c  →  x = c · (b/a)
1
Ecuación: (2/3)x = 8
2
Multiplicamos por 3/2: x = 8 · (3/2)
3
Solución: x = 12
💡 Truco: para eliminar una fracción en un coeficiente, multiplica por su inverso: (2/3)x = 8 → x = 8 · 3/2.
// sección 05

Ecuaciones mixtas

Combinan suma/resta y multiplicación/división. Estrategia: agrupar y despejar.

Estrategia general

1
Quita paréntesis aplicando la distributiva si los hay.
2
Agrupa todos los términos con x en el miembro izquierdo.
3
Pasa los números al miembro derecho.
4
Despeja x dividiendo por su coeficiente.

Ejemplo 1: ax + b = c

1
Ecuación: 2x + 5 = 13
2
Pasamos el +5: 2x = 13 − 5 = 8
3
Dividimos por 2: x = 4

Ejemplo 2: con paréntesis

1
Ecuación: 3(x − 2) = 9
2
Distributiva: 3x − 6 = 9
3
Pasamos el −6: 3x = 9 + 6 = 15
4
Dividimos por 3: x = 5

Ejemplo 3: incógnita en ambos miembros

1
Ecuación: 5x − 3 = 2x + 9
2
Agrupamos las x: 5x − 2x = 9 + 3
3
Simplificamos: 3x = 12
4
Dividimos por 3: x = 4

Ejemplo 4: con fracciones

1
Ecuación: x/2 + 3 = 7
2
Pasamos el +3: x/2 = 4
3
Multiplicamos por 2: x = 8
